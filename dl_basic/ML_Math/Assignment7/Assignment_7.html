Task 1: MAP

Consider a regression task for training data $\{ \mathbf{z} \} = \{ \mathbf{z}_1,\mathbf{z}_2,...,\mathbf{z}_n \}$, $\mathbf{z}_i = (\mathbf{x}_i,y_i)$, with data $\mathbf{x}_i \in \mathbb{R}^m$ and labels $y_i$. We assume Gaussian target noise with zero mean and variance $\sigma^2$:

$$ y_i = \mathbf{w}^\text{T}\mathbf{x}_i + \epsilon_i, \quad \epsilon_i \sim \mathcal{N}(0,\sigma^2), $$

where $\mathbf{w}$ is the parameter vector. The likelihood of data $\mathbf{z}$ to be produced by the model is

$$ p(\{\mathbf{z}\}|\mathbf{w}) = \prod_{i=1}^n p(z_i|\mathbf{w}) = \prod_{i=1}^n p(\mathbf{x}_i,y_i|\mathbf{w}) = \prod_{i=1}^n p(y_i|\mathbf{x}_i,\mathbf{w})\ p(\mathbf{x}_i). $$

The expression $p(y_i|\mathbf{x}_i,\mathbf{w})$ is the probability of the response $y_i$, given input vector $\mathbf{x}_i$ and parameter vector $\mathbf{w}$. Since $y_i - \mathbf{w}^\text{T}\mathbf{x}_i = \epsilon_i$, this is just the probability density function of $\mathcal{N}(0,\sigma^2)$ evaluated at the difference $\epsilon_i$:

$$ p(\{\mathbf{z}\}|\mathbf{w}) = \prod_{i=1}^n p(y_i|\mathbf{x}_i,\mathbf{w})\ p(\mathbf{x}_i) = \prod_{i=1}^n \frac{1}{\sqrt{2\pi\sigma^2}}\ \text{exp}\Bigl\lbrack -\frac{(y_i-\mathbf{w}^\text{T}\mathbf{x}_i)^2}{2\sigma^2} \Bigr\rbrack\ p(\mathbf{x}_i). $$

Now we assume a Gaussian prior with mean $\mathbf{\mu} = \mathbf{0} \in \mathbb{R}^n$ on $\mathbf{w} \in \mathbb{R}^n$ with unknown but fixed $\sigma_{\mathbf{w}}^2$, i.e.

$$ p(\mathbf{w}) = \mathcal{N}(\mathbf{0},\sigma_{\mathbf{w}}\mathbf{I}) = \frac{1}{(2\pi\sigma_{\mathbf{w}}^2)^{n/2}}\, \text{exp}\Bigl\lbrack -\frac{\mathbf{w}^\text{T}\mathbf{w}}{2\sigma_{\mathbf{w}}^2} \Bigr\rbrack, $$

where $\mathbf{I}$ is the identity matrix.

Task:

Show that when we do Maximum-A-Posteriori (MAP) estimation, we obtain the optimization objective (loss function) of linear regression with $L_2$ weight decay. Also, try to find an expression which is equivalent to the hyperparameter $\lambda$ that is controlling the influence of weight decay. Find an interpretation of this expression.

Calculation (50 Points):

To estimate the MAP parameters we calcualte

\begin{align*} p(\mathbf{w}|\{\mathbf{z}\}) & =\frac{p(\{\mathbf{z}\}|\mathbf{w})p(\mathbf{w})}{p(\{\mathbf{z}\})}\implies\mathbf{w_{MAP}=}\underset{\mathbf{w}}{\arg\max}p(\mathbf{w}|\{\mathbf{z}\})=\underset{\mathbf{w}}{\arg\max}p(\{\mathbf{z}\}|\mathbf{w})p(\mathbf{w})\\ \mathbf{w_{MAP}} & =\underset{\mathbf{w}}{\arg\max}\prod_{i=1}^{n}p(y_{i}|\mathbf{x}_{i},\mathbf{w})p(\mathbf{x}_{i})p(\mathbf{w})=\underset{\mathbf{w}}{\arg\max}\prod_{i=1}^{n}\frac{1}{\sqrt{2\pi\sigma^{2}}}\text{exp[}-\frac{(y_{i}-\mathbf{w}^{\text{T}}\mathbf{x}_{i})^{2}}{2\sigma^{2}}]p(\mathbf{x}_{i})\frac{1}{(2\pi\sigma_{\mathbf{w}}^{2})^{n/2}}\text{exp}[-\frac{\mathbf{w}^{\text{T}}\mathbf{w}}{2\sigma_{\mathbf{w}}^{2}}]. \end{align*}

By applying the log-trick on equation, it can be written as
\begin{align*} \mathbf{w_{MAP}=} & \underset{\mathbf{w}}{\arg\min}(-\log p(\{\mathbf{z}\}|\mathbf{w})-\log p(\mathbf{w}))\\ = & \underset{\mathbf{w}}{\arg\min}\sum_{i=1}^{n}-\log(\frac{1}{\sqrt{2\pi\sigma^{2}}})+\frac{(y_{i}-\mathbf{w}^{\text{T}}\mathbf{x}_{i})^{2}}{2\sigma^{2}}+\log p(\mathbf{x}_{i})-\log\frac{1}{(2\pi\sigma_{\mathbf{w}}^{2})^{n/2}}+\frac{\mathbf{w}^{\text{T}}\mathbf{w}}{2\sigma_{\mathbf{w}}^{2}}\\ = & \underset{\mathbf{w}}{\arg\min}\sum_{i=1}^{n}\frac{(y_{i}-\mathbf{w}^{\text{T}}\mathbf{x}_{i})^{2}}{2\sigma^{2}}+\frac{1}{2\sigma_{\mathbf{w}}^{2}}\sum_{j=1}^{m}w_{j}^{2}+const.\\ = & \underset{\mathbf{w}}{\arg\min}\frac{1}{2\sigma^{2}}\sum_{i=1}^{n}(y_{i}-\mathbf{w}^{\text{T}}\mathbf{x}_{i})^{2}+\frac{1}{2\sigma_{\mathbf{w}}^{2}}\sum_{j=1}^{m}w_{j}^{2}. \end{align*}

Then, it is obvoius that we obtain the loss function of linear regression with $L_2$ weight decay and the hyperparameter $\lambda$ that is $\frac{1}{2\sigma_{\mathbf{w}}^{2}}$. In fact, MAP try to incorporate our prior knowledge about $\mathbf{w}$ into the estimation of parameters. Thus, when the variance of $\mathbf{w}$ is high the inverse of this, $\frac{1}{2\sigma_{\mathbf{w}}^{2}}$, or $\lambda$ is low and MAP does not include the precise prior information of parameters.

# linear regression

from sklearn.linear_model import LinearRegression
def linear_regression(data, max_degree):
    
    list_of_predictors = []                             # create empty list of predictors
    list_of_coefficients = []                           # create empty list of coefficients
    predictors=['x']
    for degree in range(1, max_degree+1):               # create predictions for all polynomials with degrees 1 to max_degree
    
        if degree >=2:                                  # extend predictors with higher powers of x
            predictors.extend(['x_{}'.format(degree)])

        lin_reg = LinearRegression(normalize=True)      # linear regression
        lin_reg.fit(data[predictors],data['y'])
        y_pred = lin_reg.predict(data[predictors])
        list_of_predictors.append(y_pred)
        
        rss = sum((y_pred-data['y'])**2)                # residual sum of squares
        result = [rss]
        result.extend([lin_reg.intercept_])             # intercept, i.e. coeff. of x^0
        result.extend(lin_reg.coef_)                    # coefficents
        list_of_coefficients.append(result)
    
    return list_of_predictors, list_of_coefficients     # return predictors and coefficients (including rss)

# ridge regression

from sklearn.linear_model import Ridge

def ridge_regression(data, max_degree, alpha_ridge):
    
    list_of_predictors = []
    list_of_coefficients = []
    
    predictors=['x']
    for degree in range(2, max_degree+1):
        if degree >=2:
            predictors.extend(['x_{}'.format(degree)])
   
    for alpha in alpha_ridge:
        
# to do
        ridge_reg = Ridge(alpha, normalize=True,)      # rigid regression
        ridge_reg.fit(data[predictors],data['y'])
        y_pred = ridge_reg.predict(data[predictors])
        list_of_predictors.append(y_pred)
        
        rss = sum((y_pred-data['y'])**2)                # residual sum of squares
        result = [rss]
        result.extend([ridge_reg.intercept_])             # intercept
        result.extend(ridge_reg.coef_)                    # coefficents
        list_of_coefficients.append(result)        
# end to do    

    return list_of_predictors, list_of_coefficients
